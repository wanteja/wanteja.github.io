Object-Oriented Programming (Dart)

This assignment demonstrates my understanding of basic Object-Oriented Programming (OOP) concepts in Dart, including classes, attributes, methods, constructors, and encapsulation.

The program defines a simple class to model a real-life object (e.g., Student, Car, Product). It stores information as attributes and includes methods to perform actions on those attributes.

  • Defines a class
  • Creates objects (instances)
  • Uses methods to process or display data
  • Demonstrates encapsulation with getters/setters

ğŸ·ï¸ Example Dart Class

class Student {
  String name;
  int age;
  double cgpa;

  // Constructor
  Student(this.name, this.age, this.cgpa);

  // Method
  void displayInfo() {
    print("Name: $name");
    print("Age: $age");
    print("CGPA: $cgpa");
  }
}

  • Attributes: Variables inside a class

  • Constructor: Special function to initialize an object

ğŸ”’ Encapsulation Example

class BankAccount {
  double _balance = 0;

  // Getter
  double get balance => _balance;

  // Setter with validation
  void deposit(double amount) {
    if (amount > 0) {
      _balance += amount;
    }
  }
}
